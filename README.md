# README
## 開発言語
- Ruby2.6.5
- Rails5.2.5
## 就業Termの技術
- devise
- Ajaxを使ったコメント・お気に入り機能
## カリキュラム外の技術
- ransack
## 実行手順
```
$ git clone git@github.com:Kosuke-Nagamatsu/ritomania.git
$ cd ritomania
$ bundle install
$ rails db:create && rails db:migrate
$ rails s
```
